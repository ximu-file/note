# JSonpath 语法





语法简介

| **JsonPath**       | **描述**      |
| ------------------ | ----------- |
| $                  | 根节点         |
| @                  | 当前节点        |
| .or[]              | 子节点         |
| ..                 | 选择所有符合条件的节点 |
| *                  | 所有节点        |
| []                 | 迭代器标示，如数组下标 |
| [,]                | 支持迭代器中做多选   |
| `[start:end:step]` | 数组切片运算符     |
| ?()                | 支持过滤操作      |
| ()                 | 支持表达式计算     |
